# Habit Tracker with Gamification — UI/UX Plan ## 1. Product Vision Design a habit tracker that feels: * Rewarding (dopamine-driven) * Simple (low friction) * Addictive in a healthy way ## 2. User Experience Principles * **Reduce friction:** Logging < 2 seconds. * **Visual progress:** Instant feedback. * **Reward consistency:** 2-day grace period for streaks. ## 3. Core Screens * **Dashboard:** Greeting, Streak (🔥), Progress Ring, Quick-tap Habit List, + Add Habit. * **Add Habit Flow:** Modal with presets (Drink Water, Workout), emoji picker, frequency. * **Progress & Analytics:** Weekly heatmap, completion % chart, streak history. * **Gamification (Badges/XP):** XP bar, Levels (1-10), Grid of locked/unlocked badges. * **Profile:** Level stats and user settings. ## 4. Visual Identity * **Style:** Clean + Playful (Duolingo/Notion vibes). * **Colors:** Primary (Soft Blue/Purple), Success (Green), Streak (Orange 🔥). * **Typography:** Bold numbers, Clean sans-serif (Inter/SF Pro). * **Microinteractions:** Checkmark bounces, flame pulses, confetti.
